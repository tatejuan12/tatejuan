Project Ideas

The project that I selected to make in the future is to develop a website and possibly a smartphone app that enables the users to chat about the thousands of sports that we have on the planet, ranging from the least known to the most known sports. The app will work as a forum, and would be modeled around the idea of Reddit, but would focus on sports.

I use Reddit a lot, for several different reasons. I like chatting to people about diverse topics, however, Reddits audience is really diverse, hence the developers cannot focus on a specific demographic. I use Reddit a lot to chat about sports, most specifically football and basketball, two sports in which I am very passionate about. There I can talk to people that cheer to the same team as mine, to my opponents and share ideas and opinions. However, it is evident to me that the diverse demographic of the Reddit community makes the developers not focus on what us, the sports audience wants. For example, we really want a better UI, that facilitates the exposure of scores and information. This hasn’t happened, rather, Reddit has recently changed the UI to something that, in my opinion, is way worse.

First of all, there would be a greater control in the website. Reddit does not have any control at all, whoever wants to create a page about something can, but on our website we would have control of what subtopics would have pages created for them. For example, our team of developers would go and create a page for football, for Barcelona, for the Golden State Warriors, etc… This ecosystem is better, because this would enable us to create an environment that we want to have, and would maintain a good quality all across the board. There are countless subreddits out there that are just lazy and effortless. We would work to maintain a good environment for all of our users. We would also want to have a good public relations team in order to know what the users want. This would enable us to have a good methodology towards our UI, or, User Centered Design. However, it is important to emphasize that our team would only be in charge of the UI, the visual appearance of these subpages, they would still have people that serve as admins in order to monitor what is going on in these pages. This is important since there would be thousands of subpages, and our team would not be able to manage all of them at the same time. So, we create the pages and maintain them periodically, however, the admins of these pages are still in charge of what is going on most of the time. These admins would have the power to create discussions with more “importance”, would be able to delete comments and also kick or ban subscribers of the sport page. Moreover, we would like to have more integrated features that Reddit does not give to the user. One example is giving the user to create polls directly through the website. On Reddit the user has to go out of his way to create a poll outside of Reddit, while in our app we can just create a poll inside of it. Polls are important for the sports audience since we can know the opinion of which team/player is better, for example. The sports audience really likes to give opinions to each other. Additionally, we would like to give users the best interface possible. I believe that a good UI is the key to maintaining the user’s interest. The interface would be clean, possibly follow a materialistic design. We would include several different themes for our website, and make sure that we offer a dark option for those who like darker themes. Our website would also need to be ported well towards smartphones. I believe nowadays for a website to be used, it needs to be able to work really well on smartphones because a lot of traffic is on smartphones.

We would be making a website, hence the software that we need is just a computer to code the website. We would also need somewhere to host our servers. On the other hand, the users will need a computer with basic hardware, since going on the internet does not require a beefy hardware, and he would also need some sort of internet browser like Firefox or Google Chrome to be able to access our website.

There are several skills that would be required to accomplish the goal of creating this website. First of all, I would probably need to be able to code really well, with an emphasis on HTML, CSS, and Javascript. Other languages are also probably going to be used. It would help if I could find a crew of good coders to help me along the way, since my project is really ambitious, and for it to work I would eventually need staff. We would need servers to host our website, but that can be done through a third party medium.

If the project is successful, then I will probably be an extremely rich individual, and I will have made a large portion of the Reddit demographic happy because I do not believe that Reddit is a good place to initiate discussions in regards to sports. If successful, we would also show that a behemoth like Reddit can indeed be challenged by a new website. The sports audience would finally have a good place to discuss about their teams and sports.
